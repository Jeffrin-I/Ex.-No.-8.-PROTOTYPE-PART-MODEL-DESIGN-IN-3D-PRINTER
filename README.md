# Ex. No.   - PROTOTYPE PART MODEL DESIGN IN 3D PRINTER

### DATE: 15-03-2026
## AIM: 
### To prototype the given part model design in 3D Printer.

## REQUIREMENTS:
### ●	3D Printer machine.
### ●	CURA 4.0 Software.
### ●	Autodesk Fusion 360 Software.
### ●	System - Windows 7 or higher, 1 GB RAM.

## PROCEDURE:

### Step 1
Check all the electrical connections of the 3D printer before starting the process.

### Step 2
Install and open the 3D printing software (such as **IdeaMaker** or **Ultimaker Cura**).  
In IdeaMaker, open the software and select the **Open File** option to import the design file.


![image](https://github.com/Sellakumar1987/Ex.-No.-8.-PROTOTYPE-PART-MODEL-DESIGN-IN-3D-PRINTER/assets/113594316/059ab4e7-f3fb-49a9-ba8e-12bdd082abef)

### Step 3
Select the **Pan** option and choose the required **extruder** (left or right) for printing the design.  
Then select the **Repair** option to automatically correct any design errors in the model.

![image](https://github.com/Sellakumar1987/Ex.-No.-8.-PROTOTYPE-PART-MODEL-DESIGN-IN-3D-PRINTER/assets/113594316/835c55fd-6195-4d73-9f5c-4af36f5a4cce)

### Step 4
Use the **Move** option to adjust the position of the design on the build plate.  
Next, use the **Rotate** option to orient the model properly so that it prints easily and consumes less material.

![image](https://github.com/Sellakumar1987/Ex.-No.-8.-PROTOTYPE-PART-MODEL-DESIGN-IN-3D-PRINTER/assets/113594316/8736080c-f421-4dd0-bae8-860df6f3583e)

### Step 5
Select the **Scale** option to adjust the size of the model according to the available build plate area.  
After scaling, click the **Start** option and choose the **Standard Printing** settings.


![image](https://github.com/Sellakumar1987/Ex.-No.-8.-PROTOTYPE-PART-MODEL-DESIGN-IN-3D-PRINTER/assets/113594316/98458892-2f68-4de0-bec7-24959ec598fa)

### Step 6
Once all the settings are completed, click the **Slice** option to generate the tool path.  
Export the sliced file in **G-code format**.


![image](https://github.com/Sellakumar1987/Ex.-No.-8.-PROTOTYPE-PART-MODEL-DESIGN-IN-3D-PRINTER/assets/113594316/f4b8b55e-6cb2-46a7-b42c-180bc5e68668)

![image](https://github.com/Sellakumar1987/Ex.-No.-8.-PROTOTYPE-PART-MODEL-DESIGN-IN-3D-PRINTER/assets/113594316/eafa933a-7e03-4f73-930d-75fb28d48716)

### Step 7
After generating the **G-code file**, follow these steps to start printing:

1. Switch on the **3D printer**.
2. Insert the **USB pen drive** containing the G-code file.
3. Clean the **build plate surface** properly.
4. Load the **filament** into the required extruder.
5. Wait until the **nozzle and bed plate reach the required temperature**.
6. Close the **printer doors**.
7. Open the file from the USB storage and select the **Print** option to begin printing.
